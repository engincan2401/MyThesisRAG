

Изисквания

Преди да стартирате проекта, трябва да имате инсталирани:
1.  **Python** (3.9 или по-нов)
2.  **Ollama**: Изтеглете го от [ollama.com](https://ollama.com)
3.  Изтеглете модела Mistral чрез терминала:
    ```
    ollama pull mistral
    ```

## 📦 Инсталация

1.  Клонирайте репозиторито:

2.  Създайте виртуална среда:
    ```
    python -m venv .venv
    ```

3.  Активирайте средата:
    * Windows: `.venv\Scripts\activate`

4.  Инсталирайте зависимостите:
    ```
    pip install -r requirements.txt
    ```
